# ROMTE (Реестр перемещения технического оборудования)

ROMTE — это десктопное приложение, разработанное с использованием Python и Tkinter, для управления перемещением технического оборудования в организации. Приложение позволяет создавать запросы на перемещение, отслеживать их выполнение, а также просматривать и удалять активные запросы.

## Функциональные возможности:

*   **Просмотр технического реестра:** Отображение всей имеющейся техники с детальной информацией.
*   **Создание запроса на перемещение:** Возможность указать, какое оборудование (по ID) и куда (корпус, помещение) нужно переместить.
*   **Отслеживание статуса оборудования:** Оборудование, для которого создан запрос, получает статус "Ожидает перемещения".
*   **Выполнение запроса:** Подтверждение перемещения оборудования, изменение его местоположения и статуса на "На месте".
*   **Управление запросами:** Просмотр списка активных запросов, их удаление (отмена).
*   **Ведение журнала запросов:** Автоматическое создание записей в таблице "Запросы" при создании перемещения и их удаление после выполнения.

## Установка и запуск

### Предварительные требования:

*   **Python 3.x:** Убедитесь, что на вашем компьютере установлен Python. Скачать можно с [python.org](https://www.python.org/).
*   **Tkinter:** Обычно входит в стандартную установку Python.

### Инструкция по установке:

1.  **Клонируйте репозиторий:**
    Если вы используете Git, склонируйте репозиторий:
    ```bash
    git clone <URL_вашего_репозитория>
    cd <название_папки_репозитория>
    ```
    Если вы скачиваете файлы напрямую, просто распакуйте архив с кодом в любую папку.

2.  **Создайте базу данных (если ее нет):**
    Программа использует файл базы данных `SchoolTechnicalEquipment.db`. При первом запуске, если файл не существует, программа попытается его создать. Убедитесь, что у вас есть права на запись в той папке, где будет находиться файл базы данных.
    
    **Важно:** Перед первым запуском, или если вы впервые настраиваете базу данных, выполните SQL-команды для создания таблиц (если они еще не существуют). Вы можете сделать это с помощью любого SQLite-клиента (например, DB Browser for SQLite) или выполнив Python-скрипт с командами создания таблиц.

    *   **Таблица `технический_реестр`:**
        ```sql
        CREATE TABLE IF NOT EXISTS "технический_реестр" (
            "ID" INTEGER PRIMARY KEY AUTOINCREMENT,
            "Название" TEXT,
            "Серийный номер" TEXT,
            "Текущее местоположение" TEXT,
            "Требуемое местоположение" TEXT,
            "Статус" TEXT,
            "Последнее обновление статуса" TEXT
        );
        ```
        *Замечание: Если у вас есть другие поля в этой таблице, добавьте их соответствующие `CREATE TABLE`.*

    *   **Таблица `Запросы`:**
        ```sql
        CREATE TABLE IF NOT EXISTS "Запросы" (
            "Номер запроса" INTEGER PRIMARY KEY AUTOINCREMENT,
            "Кол-во техники" INTEGER,
            "ID техники" TEXT
        );
        ```

3.  **Запустите приложение:**
    Откройте терминал или командную строку, перейдите в папку с кодом программы и выполните:
    ```bash
    python ваш_основной_файл.py
    ```
    (Замените `ваш_основной_файл.py` на имя вашего Python-файла, например `main.py` или `app.py`).

## Использование программы

### Главное окно:

После запуска вы увидите главное окно программы:
*   **Верхняя и нижняя панели:** Информация или элементы управления.
*   **Центральная область:** Таблица `технический_реестр`, отображающая все записи.
*   **Кнопки в нижней части:**
    *   `Запрос на перемещение`: Открывает окно для создания нового запроса.
    *   `Выполнение запроса`: Открывает окно для подтверждения выполнения существующих запросов.
    *   `Просмотр запросов`: Открывает окно с таблицей всех активных запросов.
    *   `Удаление запросов`: Открывает окно для удаления выбранного запроса.
    *   `Выход`: Закрывает приложение.

### Создание запроса на перемещение:

1.  Нажмите кнопку **"Запрос на перемещение"**.
2.  В открывшемся окне:
    *   **"Оборудование (по ID)"**: Введите ID техники, которую нужно переместить. Используйте следующие форматы:
        *   `1` (один ID)
        *   `1:10` (диапазон ID от 1 до 10 включительно)
        *   `1:3, 6:10, 12` (несколько диапазонов и одиночных ID)
    *   **"Перенос в:"**:
        *   Выберите **"Корпус"** из выпадающего списка.
        *   Выберите **"Помещение"** из выпадающего списка (список помещений зависит от выбранного корпуса).
3.  Нажмите кнопку **"Подтвердить"**.
    *   Если все корректно, оборудование получит статус "Ожидает перемещения", а в таблице "Запросы" появится новая запись.

### Выполнение запроса:

1.  Нажмите кнопку **"Выполнение запроса"**.
2.  В открывшемся окне вы увидите таблицу **"Активные запросы"**.
3.  Введите **"Номер запроса"** в поле ввода.
4.  Нажмите кнопку **"Подтвердить выполнение запроса"**.
    *   Программа найдет соответствующий запрос.
    *   Для всей техники, указанной в этом запросе, "Текущее местоположение" будет обновлено на "Требуемое местоположение" из запроса.
    *   Статус техники будет изменен на "На месте".
    *   Поле "Последнее обновление статуса" будет обновлено текущей датой и временем.
    *   Запрос будет удален из таблицы "Запросы".
    *   Главная таблица `технический_реестр` и таблица запросов будут обновлены.

### Просмотр таблицы запросов:

1.  Нажмите кнопку **"Просмотр запросов"**.
2.  Откроется окно с таблицей, содержащей все активные запросы.

### Удаление запроса:

1.  Нажмите кнопку **"Удаление запросов"**.
2.  В открывшемся окне вы увидите таблицу **"Активные запросы"**.
3.  Введите **"Номер запроса"** в поле ввода, который вы хотите удалить.
4.  Нажмите кнопку **"Подтвердить удаление запроса"**.
    *   Программа найдет соответствующий запрос.
    *   Техника, связанная с этим запросом, будет возвращена к своему *текущему* местоположению (если оно было указано в запросе).
    *   Запрос будет удален из таблицы "Запросы".
    *   Таблицы `технический_реестр` и `Запросы` будут обновлены.
